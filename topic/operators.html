Mostly same as in C/C++.
The different ones are only discussed below.



Bitwise Operators:

$a and $b;
$h or $k;
$x xor $y;

These operators are same as &, | and ^ but have less precedence.


Concatenation Operator:

$a = "Hello";
$b = "World";
$message = $a." ".$b; //$message = "Hello World";

This operator concatenates the strings.


Execution Operator:

$ping = `ping -n 4 google.com`;
$copy = `copy index.html backup.html`;

The symbol is not apostrophe but a backtick and is used to execute shell commands.


Identity Operator:

$a = 333;
$b = '333';
$comparision = ($a == $b); //$comparision = TRUE
$identity = ($a === $b); //$identity = FALSE

This operator returns TRUE if and only if both the data and data type is same.
