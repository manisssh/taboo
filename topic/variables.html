No need to declare type, just use dollar/pound character before the name!


For Example:

$year = 2023;
$lang = "PHP";


Scope of Variables:

Just like in C/C++. But super global variables are accessible absolutely everywhere.
 $GLOBALS: keeps info about all user-defined global variables,
 $_COOKIES: has data input via HTTP Cookies,
 $_SESSION: has session data,
 $_ENV: has information about where PHP is running, etc
