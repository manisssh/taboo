Syntax:

function name_of_function($var1, $var2, $var3, ... $varN) {
  .
  //code
  .
  return something;
}


Special Features:

1. Use function_exits("name_of_function") to check availability.
2. Functions can return array($var1, $var2, $var3, ... $varN). 
    <?php
      $x = 4;
      $y = 6;
      $z = 5;

      add_ten($x, $y, $z);

      function add_ten($a, $b, $c) {
        $a += 10;
        $b += 10;
        $c += 10;
        return array($a, $b, $c);
      }
    ?>
3. Arguments can be passed by reference using '&' character. 
    <?php
      $x = -9;
      $y = -3;
      $z = 8;

      abs_array($x, $y, $z);

      function abs_array(&$a, &$b, &$c) {
        $a = abs($a);
        $b = abs($b);
        $c = abs($c);
      }
    ?>
4. Global variables can be pointed from inside by using "global" keyword. 
    <?php
      $a = "NePal";
      $b = "NEPALI";
      $c = "devnagari";

      fix_names();

      function fix_names() {
        global $a, $b, $c;
        $a = ucfirst(strtolower($a));
        $b = ucfirst(strtolower($b));
        $c = ucfirst(strtolower($c));
      }
    ?>
