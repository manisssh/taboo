Use define() function to declare constants.


For Example:

define("pi", "3.141592653589");
define("default", "dark-mode");


Predefined Constants:

These are ready to use constants. Ex: Magic Constants like
 __NAMESPACE__ : returns name of namespace,
 __FUNCTION__ : returns name of function,
 __CLASS__ : returns name of class,
 __METHOD__ : returns name of method, etc.
